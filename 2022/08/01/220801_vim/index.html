vim

insert

  • 키가 일반 키보드처럼 텍스트 입출력으로 동작

normal

  • 가장 중심이 되는 mode
  • esc 키로 normal mode 진입 가능
  • 키 하나하나가 명령어로 동작

command

normal mode에서 shift + ; 으로 진입

  • w (write) 저장
  • q (quit) 나가기
    • 변경사항 있으면 경고 띄워줌
    • ! (override) 이용해서 저장 없이 나갈 수 있음
      • 뭔가 잘못됐다 싶으면 꼭 q!로 탈출할 것
      • 터미널 그냥 종료하지 않기
